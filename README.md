# Selenium 4 + Pytest + POM Framework

Enterprise-grade Selenium 4 + Pytest Page Object Model automation framework  
with GitHub Actions CI/CD, Allure reporting, cross-browser support, and data-driven tests.

Built for vehicle software, wireless, and networking UI testing.

## Features
- Clean Page Object Model architecture
- Chrome + Firefox (headless in CI)
- Data-driven tests with pytest parametrize
- GitHub Actions CI/CD pipeline
- Allure + HTML reports
- Ready for vehicle dashboard / wireless / networking UI scenarios
